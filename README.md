# Console-Based To-Do List (Java)

A simple console-based To-Do List application developed using Java.  
The application allows users to add tasks and view the list of added tasks through a menu-driven interface.

## Features
- Add tasks
- View all added tasks
- Menu-driven console interface
- Lightweight and beginner-friendly design

## Concepts Used
- Java fundamentals
- Input handling using `Scanner`
- `ArrayList` for storing tasks
- Control flow using loops and conditionals

## How to Run
```bash
javac ToDoList.java
java ToDoList
